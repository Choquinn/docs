Para baixar e executar o SKlauncher, você precisará seguir esses simples passos:

## Baixando

:::aviso
O SKlauncher ainda não tem um instalador para Windows, ou qualquer outro sistema operacional.
:::

1. Baixe a versão mais recente do SKlauncher por [aqui](https://skmedix.pl/sklauncher/downloads).
2. Seleciona a versão do SKlauncher que é compatível com seu sistema operacional
    - Se você está usando Windows, baixe o arquivo .exe
    - Se vocÊ está usando Linux ou Mac OS, baixe o arquivo .jar
3. Siga as instruções para [instalar o SKlauncher](/getting-started/install)

## Erros comuns

Alguns usuários podem não estar conseguindo baixar o SKlauncher, aqui estão alguns erros comuns e como consertá-los.

### Botões de download não estão aparecendo
Há alguns motivos de o porquê de isso acontecer, mas o mais comum seria de que você está usando um bloqueador de anúncios(adblocker). Por favor desative e tente novamente.
Se ainda não funciona, tente esperar 10 minutos na tela de erro, deverá carregar após isso.

### Arquivo está sendo baixando como um arquivo .html
Isso pode acontecer se você estiver usando um gerenciador de downloads. Por favor, desative-o e tente novamente.
